# E-Shop Project

A simple e-commerce backend project built with Spring Boot and PostgreSQL.

The goal of this project is to practice backend development concepts and gradually learn technologies used in modern web applications.

## Current Features

- Product management
- Order creation
- Pagination and filtering
- Exception handling
- JWT authentication
- Docker setup
- PostgreSQL integration

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- PostgreSQL
- Hibernate / JPA
- Docker

## Running the Project

```bash
mvn clean package -DskipTests
docker compose up --build
