# 4-Week Hybrid Plan (Running + Calisthenics) — One Page

| **Day** | **Session**             | **Content**                            |
| ------- | ----------------------- | -------------------------------------- |
| Wed     | Recovery / Optional Run | 25–30 min @ 6:30–6:45/km (HR <150)     |
| Thu     | Intervals               | 4×(3 min @ 5:15–5:20 + 2 min easy)     |
| Fri     | Calisthenics Day 1      | Strength + Front Lever                 |
| Sat     | Easy Run                | 35–40 min @ 6:20–6:35/km               |
| Sun     | Progression Run         | 10 min 6:30 → 15 min 6:00 → 8 min 5:30 |
| Mon     | Calisthenics Day 2      | Skill + Handstand                      |

---

## 🔧 DAILY WARM-UP (10 min – mandatory)

* Scapula Pull-Ups: 2×10
* Band External Rotations: 2×15
* Wall Slides: 2×10
* Dead Bug: 2×10/side
* Hollow Hold: 3×20–30 sec

---

## 🟢 DAY 1 – Strength + Front Lever

* Pull-Ups: 4×6–8 (add weight if possible)
* Front Lever Holds: 5×8–15 sec
* FL Negatives/Raises: 3–4×3–6
* Dips (weighted): 4×6–10
* Hanging Leg Raises: 3×10–15

---

## 🔵 DAY 2 – Skill + Handstand

* Muscle-Ups: 5–6×1–4 (clean reps, no failure)
* Explosive Pull-Ups: 4×4–6
* Straight Bar Dips: 3×8–12
* Handstand: 10–15 min practice

  * Weeks 1–2: wall holds
  * Weeks 3–4: balance attempts
* Pike Push-Ups: 3×8–12
* L-Sit: 3–4×max hold

---

## 📈 PROGRESSION

| **Weeks 1–2**        | **Weeks 3–4**               |
| -------------------- | --------------------------- |
| Technique focus      | Add intensity               |
| Holds: 8–10 sec      | Holds: 10–15 sec            |
| Light/clean reps     | Add weight (pull-ups/dips)  |
| Muscle-ups: low reps | Muscle-ups: more consistent |

---

## ⚠️ KEY RULES

* RIR 1–2 (no failure)
* Perfect form > more reps
* Stop if shoulder pain
* Core tight (fix posture & handstand)

---

## 🎯 FOCUS CUES

* Front lever → **full body tension**
* Muscle-up → **explosive pull + smooth transition**
* Handstand → **ribs down, glutes tight (no banana)**
